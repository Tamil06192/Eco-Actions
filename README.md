# ENVIRONET

A citizen-centric web platform for environmental problem reporting and resolution.

## Prerequisites
- Node.js (v18 or higher)
- npm

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd eco-action
    ```

2.  Install dependencies (if not already done):
    ```bash
    npm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    ```

4.  **IMPORTANT**: Open your browser and visit:
    `http://localhost:5173`

    > **Do not open `index.html` directly from your file explorer.**
    > React applications require a development server to load styles and scripts correctly. If you open the file directly, you will see a blank or unstyled page.

## Features
-   **User Dashboard**: Report environmental issues with location and photos.
-   **Admin Dashboard**: Manage and update the status of reported issues.
-   **Dark Mode**: Toggle between light and dark themes.
-   **Responsive Design**: Works on desktop and mobile.
